# Changelog

All notable changes to this project will be documented in this file.

## [0.1.3] - 2024-02-19

### Added

- Support bevy 0.13.0.
- Better colors and layout.
- Configurable colors.
- Support another backspace key code.

### Removed

- Temporarily removed egui inspector until it works in latest bevy.
